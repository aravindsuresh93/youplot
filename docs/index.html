Zoom & Pan

The Zoom tool is default. Just set zoom=True.

  • Left-drag horizontally → zoom X to selection
  • Left-drag vertically → zoom Y axis
  • Double-click → reset to full range
  • 60fps regardless of dataset size
import youplot as yp fig = yp.Figure(title="Engine Temperature", zoom=True) # enables toolbar fig.line(ts_ms, temp, label="°C") fig.save("chart.html")

Crosshair Sync

All charts on a page share a cursor automatically. Each shows its own tooltip.

  • No configuration — just combine figures
  • Each chart tooltip shows its own series independently
  • Works across any number of charts
fig1 = yp.Figure(title="Temperature", zoom=True) fig1.line(ts_ms, temp, label="°C") fig2 = yp.Figure(title="Humidity", zoom=True) fig2.line(ts_ms, hum, label="%") # Crosshair sync is automatic dash = fig1 + fig2 dash.save("dashboard.html")

Region Tags

Named time windows defined in Python or drawn interactively. Appears as header band + clickable bubble. Click bubble to navigate.

  • removable=False pins permanently
  • Or draw with the Tag tool in browser
fig.tag( x_start = ts_ms[fault_start], x_end = ts_ms[fault_end], label = "Fault window", color = "#f43f5e", removable = True, ) fig.tag( x_start = ts_ms[0], x_end = ts_ms[3600], label = "Night", removable = False, # pinned )

Annotation Pins

Drop sticky notes at any x position. Defined in Python or placed interactively. Hidden when outside the zoom window. Appear as sticky notes in the panel below.

  • y_frac: 0=top, 1=bottom
  • color="" → auto-cycle palette
  • Survive HTML export
fig.pin( x = ts_ms[peak_idx], label = "Peak: 127°C", y_frac = 0.1, # near top color = "#f43f5e", ) fig.pin( x = ts_ms[event_idx], label = "System restart", y_frac = 0.5, color = "", # auto palette )

Measure Tool

Click an anchor point. As you move the cursor, a live ΔX and per-series ΔY bar appears below the chart.

  • All visible series show value difference simultaneously
  • Saved measurements persist in the chart
  • Hover saved measurement + click × to remove
# Measure tool is UI-only. # Just enable zoom=True. fig = yp.Figure(title="Signal Analysis", zoom=True) fig.line(ts_ms, signal_a, label="Ch A") fig.line(ts_ms, signal_b, label="Ch B") fig.save("analysis.html") # In browser: Measure tool → click anchor

Threshold Bands

Shade horizontal regions for normal ranges, SLA limits, or alert thresholds. Labelled, colour-coded, dashed border lines.

fig.band( y_lo = 18, y_hi = 24, label = "Comfort zone", color = "#10b981", opacity = 0.10, ) fig.hline( y = 24, label = "Upper limit", color = "#f97316", dash = True, )

Quickstart

Python in. HTML out.

quickstart.py
import  youplot as yp

ts_ms = [1_700_000_000_000 + i * 1000 for i in range(86_400)]  # unix ms

fig = yp.Figure(title="24h Temperature", zoom=True, legend=True)
fig.line(ts_ms, temp, label="°C", color="#f97316", fill=True)
fig.band(y_lo=18, y_hi=24, label="Comfort", color="#10b981")
fig.vline(x=ts_ms[3600], label="Event", color="#dc2626")
fig.save("temperature.html")   # open in any browser
dashboard.py — synced multi-chart
fig1 = yp.Figure(title="Engine", height=380, zoom=True)
fig1.line(ts_ms, coolant, label="Coolant °C", color="#ef4444")
fig1.line(ts_ms, oil,     label="Oil °C",     color="#f97316", dash=True)
fig1.tag(x_start=fault_start, x_end=fault_end, label="Fault")
fig1.pin(ts_ms[peak_idx], label="Peak: 127°C", y_frac=0.1)

fig2 = yp.Figure(title="RPM & Fuel", height=280, zoom=True)
fig2.line(ts_ms, rpm,  label="RPM",      color="#6366f1")
fig2.line(ts_ms, fuel, label="Fuel PSI", color="#16a34a")

dash = fig1 + fig2  # or yp.combine(fig1, fig2)
dash.save("engine.html")

Installation

Requires Python 3.10+. No runtime dependencies.

$ pip install youplot

The uPlot JS runtime (50 KB) is bundled. Generated HTML files work offline and open from file://.

Concepts

  • A Figure is one chart — series, annotations, axes.
  • A Dashboard groups figures into one HTML page with shared crosshair sync.
  • Everything renders to a self-contained HTML string or file. No server required to view.
  • X data is a list of numbers. For timeseries use Unix milliseconds: int(time.time()) * 1000.

Timeseries auto-detected when x values exceed 1e11. Axis labels and tooltips format automatically.

Figure()

fig = yp.Figure(title="My Chart", subtitle="Optional", theme="light", height=400, width=None, y_label="Signal", zoom=True, legend=True)

All methods return self for chaining.

line()

fig.line(
  x, y,               # required: lists of numbers
  label        = "",
  color        = None,  # hex, named, or auto
  width        = 2.0,
  dash         = False,  # True or [on_px, off_px]
  fill         = False,  # gradient fill under line
  fill_opacity = 0.15,
  points       = False,
  points_size  = 4.0,
  step         = False,  # staircase rendering
  gap_threshold= None,
  hover_unit   = "",    # e.g. " °C"
  hover_format = "",    # e.g. ".2f"
)

scatter()

fig.scatter(x, y, label="", color=None,

  jitter_x=0.0, jitter_y=0.0)

band()

Horizontal shaded band for normal ranges, SLA limits, alert zones.

fig.band(y_lo=18, y_hi=24, label="Comfort", color="#10b981", opacity=0.12)

vline() / hline()

fig.vline(x=ts_ms[idx], label="Event", color="#dc2626")
fig.hline(y=100, label="Max", color="#f97316", dash=True)

For timeseries vline, pass x in unix milliseconds.

region()

fig.region(x_start=ts_ms[0], x_end=ts_ms[3600],
         label="Night", color="indigo", opacity=0.06)

tag()

fig.tag(x_start=ts_ms[a], x_end=ts_ms[b],
      label="Fault", color="#f43f5e", removable=True)

pin()

fig.pin(x=ts_ms[idx], label="Peak", y_frac=0.1, color="")
ParamTypeDescription
xfloatX position. Unix ms for timeseries.
labelstrText on popup and sticky note.
y_fracfloatVertical position. 0=top, 1=bottom. Default 0.2.
colorstrColour. Empty = auto-cycle palette.

save() / show() / to_html()

fig.save("out.html")      # write to disk
fig.show()                # open in browser
html = fig.to_html()    # return string
frag = fig.to_fragment() # no <head>/<body>

combine()

dash = fig1 + fig2 + fig3
dash = yp.combine(fig1, fig2, title="Dashboard")
dash = yp.Dashboard().add(fig1).add(fig2)
dash.save("out.html")

Zoom & Pan (UI)

  • Left-drag horizontally → zoom X.
  • Left-drag vertically → zoom Y.
  • Double-click → reset.

Tags (UI)

Switch to Tag tool. Drag to define region, enter name. Click bubble to navigate.

Annotations (UI)

Switch to Annotate. Click to drop pin, enter label. Pin hidden when outside zoom window. Sticky notes panel below. Click note to navigate.

Measure (UI)

Switch to Measure. Click anchor → live ΔX and ΔY per series. Hover saved measurement + × to remove.

Export (UI)

↓ Export button downloads self-contained HTML. Tags and pins preserved. Opens in any browser.

Colors

  • Hex: "#f43f5e"
  • Named: "rose" "indigo" "emerald" "orange" "violet" "cyan" "amber" "pink" "lime" "teal"
  • None — auto-cycles per Figure
