Crosshair Sync

All charts on a page share a live cursor automatically. Move over one panel and every other follows instantly — no configuration needed. Each chart shows its own series values independently in the tooltip.

  • Works across any number of combined figures
  • Each tooltip is independent per chart
  • No JS needed — just fig1 + fig2
  • Essential for correlating signals across panels
# Crosshair sync is automatic when you combine fig1 = up.Figure(title="Temperature", zoom=True) fig1.line(ts_ms, temp, label="°C") fig2 = up.Figure(title="Humidity", zoom=True) fig2.line(ts_ms, hum, label="%") fig3 = up.Figure(title="Pressure", zoom=True) fig3.line(ts_ms, pres, label="hPa") # All three cursors stay in sync automatically dash = fig1 + fig2 + fig3 dash.save("dashboard.html")

Region Tags

Name a time window in Python and it renders as a coloured header band with a clickable bubble. Draw new ones interactively in the browser. Perfect for marking fault windows, deploy phases, or test runs before sharing the file with a teammate.

  • removable=False pins the tag permanently
  • Draw new tags interactively with the Tag tool
  • Click a bubble to jump to that region
  • Preserved on HTML export
fig.tag( x_start = ts_ms[fault_start], x_end = ts_ms[fault_end], label = "Fault window", color = "#f43f5e", removable = True, ) fig.tag( x_start = ts_ms[0], x_end = ts_ms[3600], label = "Baseline", color = "#6366f1", removable = False, # pinned )

Annotation Pins

Drop sticky notes at any X position, defined in Python or placed interactively. Pins hide when you zoom past them, reappear when you zoom back. A panel below lists all active annotations — click any note to scroll to it.

  • y_frac: 0 = top of chart, 1 = bottom
  • color="" auto-cycles the palette
  • Survive HTML export — share with teammates
fig.pin( x = ts_ms[peak_idx], label = "Peak: 127°C", y_frac = 0.1, # near top color = "#f43f5e", ) fig.pin( x = ts_ms[restart_idx], label = "System restart", y_frac = 0.5, color = "", # auto palette )

Measure Tool

Click an anchor point in the chart. As you move the cursor, a live ΔX and per-series ΔY bar appears below. All visible series show their difference simultaneously. Saved measurements persist in the exported HTML.

  • All visible series show ΔY simultaneously
  • Saved measurements stay in the exported file
  • Hover a saved measurement + click × to remove
  • Great for checking rise times, durations, deltas
# Measure is UI-only. Enable zoom=True # to show the toolbar in the browser. fig = up.Figure( title = "Signal Analysis", zoom = True, legend = True, ) fig.line(ts_ms, signal_a, label="Ch A") fig.line(ts_ms, signal_b, label="Ch B") fig.save("analysis.html") # In browser: Measure → click anchor point

Threshold Bands

Shade horizontal regions for normal ranges, SLA limits, or alert thresholds. Combine with hline() for crisp boundary markers. Everything is labelled and colour-coded.

  • Shade alert zones, comfort ranges, or SLA windows
  • Combine with hline() for the boundary line
  • Works with any Y-axis unit
fig.band( y_lo = 18, y_hi = 24, label = "Comfort zone", color = "#10b981", opacity = 0.10, ) fig.hline( y = 24, label = "Upper limit", color = "#f97316", dash = True, ) fig.band( y_lo = 35, y_hi = 100, label = "Critical zone", color = "#f43f5e", opacity = 0.07, )

Installation

Requires Python 3.10+. No runtime dependencies.

$ pip install youplot

The uPlot JS runtime (50 KB) is bundled. Generated HTML files work offline and open from file://.

How it works

  • A Figure is one chart — series, annotations, axes.
  • A Dashboard groups figures into one HTML page with shared crosshair sync.
  • Everything renders to a self-contained HTML string or file. No server required.
  • X data is a list of numbers. For timeseries use Unix milliseconds: int(time.time()) * 1000.

Timeseries is auto-detected when x values exceed 1e11. Axis labels and tooltips format automatically.

New Figure

fig = up.Figure(
    title    = "My Chart",
    subtitle = "Optional subtitle",
    theme    = "light",
    height   = 400,
    width    = None,       # None = fill container
    y_label  = "Signal",
    zoom     = True,
    legend   = True,
)

All methods return self for chaining.

Line series

fig.line(
    x,                       # list of numbers (unix ms for timeseries)
    y,                       # list of numbers
    label        = "",
    color        = None,    # hex, named, or auto-cycle
    width        = 2.0,
    dash         = False,  # True or [on_px, off_px]
    fill         = False,  # gradient fill under the line
    fill_opacity = 0.15,
    points       = False,
    points_size  = 4.0,
    step         = False,  # staircase rendering
    gap_threshold= None,
    hover_unit   = "",    # e.g. " °C"
    hover_format = "",    # e.g. ".2f"
)

Named time windows that appear as a coloured band with a clickable bubble. Define in Python or draw interactively in the browser.

fig.tag(

    color     = "#f43f5e",

Measure tool

UI-only. Enable the toolbar with zoom=True, then click the Measure tool in the browser. Click an anchor point and move the cursor to see live ΔX and ΔY for all visible series. Saved measurements persist in the exported HTML.

Zoom & pan

  • Left-drag horizontally → zoom X to selection.
  • Left-drag vertically → zoom Y axis.
  • Double-click → reset to full range.

Save, show, export

fig.save("out.html")        # write to disk
fig.show()                  # open in default browser
html = fig.to_html()      # return full HTML string
frag = fig.to_fragment()   # no <head>/<body> wrapper

Combine charts

# Three equivalent ways to combine
dash = fig1 + fig2 + fig3
dash = up.combine(fig1, fig2, title="Dashboard")
dash = up.Dashboard().add(fig1).add(fig2)
dash.save("out.html")
